Final Project Proposal — RubberbandTurretinator

Illustration placeholder — simple labeled sketch of rotating base, ultrasonic sensor, and servo-based rubber band launcher.

Project Concept / Motivation

RubberbandTurretinator: I’m going to make a rotating turret that continuously scans. When an ultrasonic sensor detects a close object it stops rotation, launches a rubber band (missile) and releases to fire at the target, then resumes scanning for enemies.

Anticipated Bill of Materials

  • Arduino Uno
  • Stepper motor (360 rotation) + motor driver
  • External power for DC motor (battery pack, etc)
  • Servo motor for firing mechanism
  • Ultrasonic sensor (HC-SR04)
  • Rubberbands + 3D-printed or laser-cut turret mount
  • Hot glue for adhesive
  • Wirings, resistors, diodes

Timeline & Contingency Plans

  1. Build turret base and mount motor + motor driver, integrate the sensor
  2. Figure out the firing mechanism, tune the distances & timings, make some photos and get the documentation started
  3. Final polish, slides, work on page..

If something goes wrong

  • If DC motor/power issues → use servo motor
  • If servo strength is insufficient → switch to a stronger servo or design a lever with mechanical advantage.
