How many LEDs can blink independently? How much current?

  • Pins available: Arduino Uno has 14 digital pins (D0–D13) + 6 analog pins usable as digital (A0–A5) → up to 20 outputs. Pins D0 (RX) and D1 (TX) are used for the USB serial connection (uploading code and Serial Monitor). If you rely on serial communication, avoid D0/D1, leaving ~18 practical pins.
  • Current limits: Design for ≤ ~20 mA per pin and ≤ ~200 mA total. I target ~10 mA per LED for brightness + safety.
  • Example totals:
    • 16 LEDs × 10 mA ≈ 160 mA (comfortable under ~200 mA)
    • 18 LEDs × 10 mA ≈ 180 mA (still under, less headroom)
  • Conclusion: Practically, about ~16 independently blinking LEDs at ~10 mA each is a safe target on an Uno without extra hardware.

Q3

If only one LED is ON at once: The limit is no longer current draw, but pin count. An Arduino Uno has up to ~20 pins (D0–D13 plus A0–A5). Pins D0 (RX) and D1 (TX) handle USB serial (uploads and Serial Monitor), so if you need serial communication, avoid using them; that leaves ~18 LEDs you can still control independently. Since only one LED is lit at a time, the current is just that single LED’s draw (≈10 mA). This means you can support the full set of ~18 LEDs safely when only one is active at once.
